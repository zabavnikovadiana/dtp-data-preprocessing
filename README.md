# dtp-data-preprocessing
Предобработка данных о ДТП в Российской Федерации в 2018 году.
Была выполнена очистка и подготовка данных dtp_last.csv, содержащего информацию о ДТП в Российской Федерации за 2018 год, к дальнейшему анализу. Мы изучили общую информацию датасета, осуществили предобработку заголовков, заполнили пропуски, нашли проблемы в данных, преобразовали типы данных, удалили дубликаты, экспортировали результаты в excel.
